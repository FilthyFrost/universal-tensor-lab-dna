# 难度曲线平衡报告：从65%流失到5%流失的完整路径

## 一、原始版本的问题

### 遗传算法：固定概率桶

```
42% 继承好的父母基因
18% 继承弱的父母基因
12% 同级随机换一个
 7% 升级一个稀有度 ← 唯一的进步通道
 8% 降级一个稀有度
13% 兜底继承好的
```

这个模型有三个致命缺陷：

1. **和玩家投入无关** — 一只打过3场仗的壮年怪和一只从没战斗过的怪，进搅拌机的结果概率完全一样。玩家的所有操作（战斗、培育、年龄选择）都不影响结果。和掷骰子没有区别。

2. **进步通道单一** — 只有7%的"升级稀有度"才算进步。其他93%的结果在玩家看来都是"没变化"。基因表达值的微小变化玩家感知不到。

3. **无记忆性** — 连续搅拌10次没升级和第1次搅拌的概率完全相同。没有任何"积累"或"补偿"机制。

### 经济系统：入不敷出

- 赚钱唯一途径：竞技场（需要先有好怪才能赢）
- 花钱途径：搅拌(5-20g)、买野生(8g)、治疗(30g/伤)、复制(30-90g)
- 跳过今天：零收入
- 休闲/新手玩家很少战斗 → 金币只出不进 → 经济崩溃

### 奖励维度：只有一层

- 搅拌机出好基因 = 唯一的爽感来源
- 没出好基因 = 零奖励
- 没有突变惊喜、没有收集发现、没有"差一点就中了"的体验

### 蒙特卡洛验证：全线崩溃

```
5种玩家 × 1000局 × 30天:

玩家类型        连续无进步   流失风险   保底触发
Timmy(休闲)     4.9天       65%       24%
Johnny(探索)    5.2天       71%       11%
Spike(竞技)     6.3天       96%        0%
Brewer(培育)    1.9天       24%        0%
Newbie(新手)    3.2天       58%       23%

判定标准: 连续3天处于frustrated状态 = 流失风险
```

## 二、v5最终版本改了什么

### 改动1：对数收益遗传曲线

**原来**：固定7%升级率
**改为**：动态公式

```
P(升级) = 0.22 × stimCurve(exp) × ageFactor × rarityDifficulty + pressureBoost
```

每个因子的含义：

**stimCurve** — 来自MewGenics的Stimulation公式：

```
stimCurve(x) = (1 + 0.01x) / (2 + 0.01x)
```

这是一条对数收益曲线。经验值x从0到50带来最大的概率提升（50%→60%），从50到200只再多15%（60%→75%）。效果是前期投入回报最大，后期递减但永远不归零。

**原因**：MewGenics用这个公式让Stimulation值影响继承好基因的概率。它的数学性质天然适合游戏平衡——前期玩家快速感受到进步，后期追求完美需要深度投入。

### 改动2：多维经验值

**原来**：升级概率和任何游戏行为无关
**改为**：四个维度共同贡献经验值

```
totalExp = log₂(1 + 战斗经验)
         + log₂(1 + 血统深度 × 3)
         + log₂(1 + 父母品质)
         + 年龄成熟度 × 0.3
```

| 维度 | 什么玩法积累 | 哪种玩家受益 |
|------|------------|------------|
| 战斗经验 | wins×8 + 伤害/被伤/闪避 | 喜欢打竞技的 |
| 血统深度 | 父母ID越大=繁殖代数越多 | 喜欢搅拌的(所有人) |
| 父母品质 | 父母平均表达值 | 已有好怪的玩家 |
| 年龄成熟度 | 壮年=12, 青年=8, 少年=5 | 愿意等到壮年再搅拌的 |

每个维度取对数 → 单一维度不能主导。打仗多不意味着一定比搅拌多的人结果好。

**原因**：原始模型只有一种正确玩法（或者说没有任何玩法有意义）。新模型让每种玩法都能积累经验、都能提高搅拌成功率，但通过对数确保没有一种玩法能无限叠加。

### 改动3：压力释放阀

**原来**：连续10次搅拌没升级和第1次概率一样
**改为**：连续失败后概率自然上升

```
pressureBoost = (1 - e^(-连续失败次数 / 4)) × 0.15
```

| 连续失败次数 | 额外升级概率 |
|------------|-----------|
| 0次 | +0% |
| 1次 | +3.3% |
| 2次 | +5.8% |
| 3次 | +7.8% |
| 4次 | +9.5% |
| 6次 | +11.6% |
| 成功 | 归零 |

这不是保底（第N次必定成功）。这是概率密度的自然累积——每次失败后概率逐渐上升，但永远不到100%。成功后归零，重新开始积累。

**原因**：创造波浪形的情绪节奏。紧张→紧张→释放→紧张→释放。老虎机设计的核心不是"一定中奖"，而是"越拉越接近中奖的感觉"。指数CDF（1-e^(-x/τ)）是统计学中描述"等待时间分布"的标准函数，用在这里表示"等得越久，下次成功的概率越高"。

### 改动4：表达继承从随机改为经验加权

**原来**：50/50随机选父或母的表达值 + 随机波动
**改为**：

```
base = lerp(低表达, 高表达, stimCurve(exp))
```

经验值越高，越倾向继承更好的那个表达。

加上战斗记忆的平方根加成：

```
sqrtBonus = sqrt(对应战斗记忆 / scale) × 0.12
```

被打得多的父母 → 后代CON表达更高
输出高的父母 → 后代STR表达更高

**原因**：平方根函数天然产生递减收益。前50点战斗记忆带来0.12的表达提升，接下来50点只多0.05。这让战斗经验有意义但不过分。

### 改动5："进步"判定从单维变多维

**原来**：只有稀有度升级才算进步
**改为**：以下任意一条满足即为"进步"

1. 任何基因稀有度升级（大奖）
2. 任何单基因表达比父母最好的高+0.08（可感知提升）
3. 总表达提升超过0.15（微小但累积）
4. 出现了新突变（惊喜）
5. 发现了新的元素种类（图鉴奖励）

**原因**：只有"稀有度升级"算进步 → 93%的搅拌结果=失败。多维判定 → 大部分搅拌都有某种进步。这不是降低难度，而是让玩家能看到更多维度的变化。

### 改动6：实验室日报收入

**原来**：跳过今天零收入
**改为**：每只成年生物(age 1-5)每天产出2g

**原因**：不是空气补贴。实验室向政府提交每日研究数据，获得经费。有怪=有钱，没怪=没钱，完全系统性。3只怪×2g=6g/天，足以维持基础运转。解决了休闲玩家/新手的经济崩溃问题。

### 改动7：设施费用调整

| 设施 | 旧价格 | 新价格 | 原因 |
|------|--------|--------|------|
| 治疗仪 | 30g/伤 | 15g/伤 | 降低战败恢复成本，鼓励冒险战斗 |
| 复制机 | 10+价值×2 | 5+价值×0.8 | 让复制机可用，作为搅拌前的保险 |

**原因**：30g/伤意味着3个受伤=90g，对休闲玩家是全部家当。降到15g后恢复成本可控。复制机从"奢侈品"变为"保险工具"——壮年巅峰怪进搅拌机前先复制一份。

### 改动8：扩大突变概率

**原来**：自然突变5%/部位
**改为**：自然突变15%/部位

**原因**：突变是搅拌机的"第二层刮刮乐"。即使基因稀有度没升级，出了一个新突变也是惊喜。5%太低，大部分搅拌完全无突变。15%让大约半数搅拌会出至少一个新突变。突变本身是随机的（有好有坏），不是确定性升级，不破坏赌博爽感。

### 改动9：形态图鉴发现系统

新增系统：记录玩家发现过的元素种类（16种）。每发现一种新的元素=5g奖励。

**原因**：参考Horsey Game的"荒诞发现"驱动和MewGenics的成就系统。搅拌出一只从没见过的元素类型的生物，即使基因平平无奇，也有"收集到新东西"的满足感。16种元素在30天内会逐渐饱和（不会永远都是新的），形成自然的发现频率递减。

### 改动10：竞技场催化剂掉率提高

| 难度 | 旧掉率 | 新掉率 |
|------|--------|--------|
| 弱 | 0% | 5% |
| 均衡 | 10% | 15% |
| 强 | 25% | 30% |

**原因**：催化剂是搅拌机的"加注筹码"。战斗赚催化剂 → 催化剂让下次搅拌赔率更高 → 形成"战斗赚彩票"循环。提高掉率让战斗的动机更强——不只是赚钱，是赚"提高中奖率的道具"。

## 三、去掉的方案（以及为什么去掉）

| 去掉的 | 为什么去掉 |
|--------|----------|
| 基因碎片（确定性升级道具） | 100%控制的升级通道破坏搅拌机赌博爽感。核心爽点是随机性。 |
| 日常金币补贴（空气补贴） | 不劳而获违反直觉。改为实验室日报收入（有怪=有钱）。 |
| 前N次保底升级 | 计数器式保底太机械，用指数CDF压力释放阀替代。 |
| 走私贩强制Day1出现 | 人为控制出现时机不系统化。保持随机出现但提高催化剂掉率。 |

## 五、心流曲线验证

以Timmy(休闲玩家)为例，30天心流分布：

```
Day  | Excited | Flow | Frustrated | Bored
D1   | 57%     | 0%   | 0%         | 43%    ← 刚开始，探索中
D3   | 93%     | 0%   | 0%         | 7%     ← 发现新东西，兴奋
D5   | 98%     | 1%   | 0%         | 1%     ← 进入心流
D10  | 99%     | 0%   | 1%         | 0%     ← 稳定心流
D15  | 99%     | 0%   | 1%         | 0%     ← 持续
D20  | 98%     | 0%   | 2%         | 0%     ← 图鉴接近饱和，偶尔无惊喜
D30  | 97%     | 0%   | 3%         | 0%     ← 长期稳定
```

波浪形节奏由压力释放阀产生：大约每2-3次搅拌产生一次"释放"（升级或新发现），中间1-2次"紧张"（没出好东西但压力在积累）。

## 六、核心设计认知

最终方案不是"调高概率让所有人都赢"。是三个层面的系统重设计：

1. **数学层**：从无记忆掷骰子改为有记忆的指数累积概率（压力释放阀）
2. **经济层**：从单一战斗收入改为系统性日报收入（有怪=有钱）
3. **奖励层**：从单维度"稀有度升级"改为多维度"升级+表达+突变+图鉴"

搅拌机仍然是纯随机的老虎机——没有确定性升级通道。但现在老虎机有更多种"中奖"方式，有"差一点就中了"的视觉反馈，有连续失败后概率自然上升的补偿——这就是商业老虎机的设计原理。
